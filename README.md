# Multi-Form

This is a simple implementation of a multi-step form, designed to collect complex input in a clean, organized, and user-friendly way. It breaks down large forms into smaller, manageable stepsâ€”improving usability and reducing user friction.

## Features

- Multi-step navigation with form validation
- Easily extendable form schema
- Built-in support for various field types (text, select, date, boolean, etc.)
- State management for collected input across steps

## Use Cases

- Signup flows
- Onboarding processes
- Complex data input (e.g., applications, surveys, checkout)
